Guia de Estudos de Análise de Dados Exploratória

Este guia interativo é um compilado de estudos sobre os conceitos fundamentais da Análise de Dados Exploratória. O objetivo é apresentar os principais tópicos, desde a representação de distribuições até a aplicação de modelos de regressão, de forma clara, prática e visual.

Começar a Explorar

Glossário Interativo

Consulte rapidamente o significado dos termos mais importantes da Análise de Dados. Clique em um conceito para expandir e ver a definição.

Análise Exploratória (ADE)
Processo inicial de análise para resumir as principais características de um dataset, frequentemente usando visualizações gráficas para encontrar padrões e anomalias.
Variável Categórica
Classifica indivíduos em grupos. Nominal: categorias sem ordem (ex: Gênero). Ordinal: categorias com uma ordem lógica (ex: Escolaridade - fundamental, médio, superior).
Variável Quantitativa
Assume valores numéricos. Discreta: valores contáveis e finitos (ex: Nº de cilindros). Contínua: pode assumir qualquer valor em um intervalo (ex: Altura, Peso).
Média, Mediana, Moda
Média: a soma de todos os valores dividida pelo número de observações. Sensível a outliers. Mediana: o valor central de um conjunto de dados ordenado. Robusta a outliers. Moda: o valor que aparece com mais frequência.
Desvio-Padrão
Medida de dispersão que indica o quão espalhados os dados estão em relação à média. Um desvio-padrão baixo indica que os pontos de dados tendem a estar próximos da média.
Outlier
Um ponto de dado que se distancia significativamente dos outros. Outliers podem distorcer a média, correlação e a reta de regressão.
Distribuição Normal (Gaussiana)
Uma distribuição de probabilidade simétrica em forma de sino, definida por sua média (μ) e desvio-padrão (σ). Muitos fenômenos naturais seguem este padrão.
Diagrama de Dispersão
Um gráfico que usa pontos para representar os valores de duas variáveis numéricas, revelando a força e a direção da relação entre elas.
Correlação (r)
Uma medida numérica (de -1 a +1) que quantifica a força e a direção da relação *linear* entre duas variáveis. Próximo a 0 significa fraca, próximo a 1 ou -1 significa forte.
Regressão Linear
Um método estatístico para modelar a relação entre uma variável dependente (y) e uma variável independente (x), ajustando uma linha reta aos dados.
Variável Explicativa e Resposta
A Explicativa (x) é a variável que se acredita influenciar a outra. A Resposta (y) é a variável que é medida ou observada como resultado.
Coeficiente de Determinação (R²)
Indica a porcentagem da variação na variável resposta (y) que pode ser explicada pelo modelo de regressão. Um R² de 0.80 significa que 80% da variação é explicada.

1. Representação de Distribuições

A base da análise de dados começa em como organizamos e visualizamos a informação. Aqui, exploramos os componentes fundamentais de um conjunto de dados e as formas de representá-los.

Indivíduos e Variáveis

Um conjunto de dados (dataset) é composto por indivíduos (os objetos descritos, ex: pessoas, produtos) e variáveis (as características desses indivíduos, ex: idade, preço).

Tipos de Variáveis

As variáveis podem ser Categóricas (classificam em grupos, ex: gênero) ou Quantitativas (valores numéricos, ex: altura). Entender essa diferença é crucial para escolher o gráfico certo.

Medidas Numéricas

Resumimos os dados com números. Medidas de centro (média, mediana) nos dizem onde os dados se concentram, e medidas de dispersão (desvio-padrão) nos dizem o quão espalhados eles estão.

10 Tipos de Gráficos Essenciais

1. Gráfico de Barras

Compara valores entre diferentes categorias. Ideal para dados categóricos.

2. Gráfico de Linhas

Mostra tendências ou mudanças ao longo do tempo. Conecta uma série de pontos de dados.

3. Gráfico de Pizza

Exibe a proporção de cada categoria em relação a um todo. Melhor para poucas categorias.

4. Histograma

Representa a distribuição de frequência de dados quantitativos contínuos.

5. Diagrama de Dispersão

Mostra a relação entre duas variáveis quantitativas. Ótimo para ver correlações.

6. Boxplot

Resume a distribuição de dados através de cinco números (mínimo, Q1, mediana, Q3, máximo).

7. Mapa de Calor

Usa cores para representar a magnitude dos dados em uma matriz, útil para visualizar correlações.

8. Gráfico de Área

Similar ao gráfico de linhas, mas com a área abaixo da linha preenchida para indicar volume.

9. Gráfico de Bolhas

Um diagrama de dispersão que adiciona uma terceira dimensão: o tamanho da bolha.

10. Ramo e Folhas

Apresenta dados quantitativos de forma a manter os valores individuais, similar a um histograma.

2. As Distribuições Normais

Muitos fenômenos no mundo real seguem um padrão previsível conhecido como Distribuição Normal. Entender este conceito é fundamental para a estatística inferencial.

A Curva em Forma de Sino

A Distribuição Normal, ou Curva de Gauss, é simétrica e tem um formato de sino. Ela é completamente definida por sua média (μ) e seu desvio-padrão (σ).

A Regra 68-95-99,7

Esta regra empírica nos diz que, em uma distribuição normal, aproximadamente 68% dos dados estão a 1 desvio-padrão da média, 95% a 2 desvios-padrão, e 99,7% a 3 desvios-padrão.

Escore-Z (Padronização)

O Escore-Z nos diz a quantos desvios-padrão um ponto de dado está da média. Isso nos permite comparar valores de diferentes distribuições e calcular probabilidades.

// Exemplo: Nota de um aluno
let nota = 7;
let mediaTurma = 5;
let desvioPadrao = 2;
let z = (nota - mediaTurma) / desvioPadrao;
// z = 1.0. A nota está 1 desvio-padrão acima da média.

3. Diagramas de Dispersão

Como podemos visualizar a relação entre duas variáveis quantitativas? O diagrama de dispersão é a nossa principal ferramenta para investigar associações e padrões.

Explicativa vs. Resposta

Em uma análise, a variável explicativa (x) é aquela que pensamos que influencia a variável resposta (y). Por exemplo, horas de estudo (x) podem influenciar a nota na prova (y).

Associação e Direção

A associação pode ser positiva (quando x sobe, y também sobe) ou negativa (quando x sobe, y desce). A força da associação depende de quão próximos os pontos estão de uma linha imaginária.

Cuidado: Associação ≠ Causa

É crucial lembrar que uma associação forte não prova uma relação de causa e efeito. Muitas vezes, uma terceira variável oculta pode estar influenciando ambas.

4. Correlação

O coeficiente de correlação (r) nos dá um número exato para medir a força e a direção de uma relação *linear* entre duas variáveis quantitativas.

O Coeficiente (r)

O valor de 'r' varia de -1 a +1.
• r = +1: Correlação linear positiva perfeita.
• r = -1: Correlação linear negativa perfeita.
• r = 0: Nenhuma correlação linear.

Propriedades de 'r'

A correlação não tem unidade de medida e não muda se trocarmos os eixos x e y. Ela é fortemente afetada por outliers e mede apenas a força de relações *lineares*.

Quarteto de Anscombe

Este famoso exemplo mostra quatro conjuntos de dados com estatísticas idênticas (média, desvio-padrão, correlação), mas com gráficos totalmente diferentes. Isso prova que sempre devemos visualizar nossos dados!

5. Regressão Linear

Se a correlação nos diz "se" existe uma relação linear, a regressão nos diz "qual" é essa relação, permitindo-nos criar um modelo para fazer previsões.

A Reta de Melhor Ajuste

A regressão linear encontra a única linha reta que melhor se ajusta aos dados. A equação dessa reta é ŷ = a + bx, onde 'b' é a inclinação e 'a' é o intercepto.

Método dos Mínimos Quadrados

Este é o método matemático usado para encontrar a reta de regressão. Ele funciona minimizando a soma dos quadrados dos erros (as distâncias verticais entre cada ponto e a reta).

Coeficiente de Determinação (R²)

R² nos diz qual porcentagem da variação na variável resposta (y) pode ser explicada pela variável explicativa (x). Um R² de 0.85 significa que 85% da variação é explicada pelo modelo.

Fazendo Previsões

Uma vez que temos a equação da reta, podemos usá-la para prever o valor de 'y' para um novo valor de 'x'.

// Ex: Vendas de sorvete = -159.47 + 30.08 * Temperatura
let temperatura = 25; // Prever vendas para 25°C
let vendasEstimadas = -159.47 + 30.08 * temperatura;
// vendasEstimadas ≈ R$ 592,53
